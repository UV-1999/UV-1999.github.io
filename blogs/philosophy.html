My unwarranted notes on philosophy

Philosophy - The study of reality (including agents) and the relationships of agents with reality.

Reality - Everything (entities and properties).

Agent - Agents are entities that are able to experience reality and do reasoning. Reasoning is the process through which the agent forms beliefs. A belief is the attitude (perception) of an agent towards any entity or property. All philosophical inquiry is conducted by an agent.

Metaphysics - The study of reality.

Logic - The formal language (communication and description) of philosophy.

Propositions are beliefs expressed in some language. "True" and "False" are the two allowed logical values that a proposition can take. Propositions are either true or false, nowhere in between (Law of excluded middle).

Logical Connectives

Logical connectives are fundamental operations that act on proposition(s) to develop compound propositions. They are:

Negation (not) - Every proposition has a corresponding negation (opposition). The negation of true is false and vice versa. if P is true then ¬ P is false.

Disjunction (or) - Connects propositions with "or". (P ∨ Q). True when at least one proposition is true. Disjunction is commutative and associative.

Conjunction (and) - Connects propositions with "and". (P ∧ Q). True only if all propositions are true. Conjunction is commutative, associative and distributive over disjunction.

Once having determined the truth values of atomic propositions, the truth value of the compound proposition (propositions composed of some atomic propositions) can be determined using the definitions of the logical connectives. This process is called a logical proof. If a compound proposition that is always true regardless of the truth values of the atomic propositions it contains, is called a tautology. Similarly, a compound proposition that is always false is called a contradiction. A compound proposition that is neither a tautology nor a contradiction is referred to as a contingency. The truth value of a contingent proposition hence depends on the atomic propositions.

De Morgan’s Laws

First Law: ¬ (P ∧ Q) = ¬P ∨ ¬Q
¬ (P ∧ Q) is true when P or Q is false.
This means at least one of ¬P or ¬Q must be true.
Therefore, ¬ (P ∧ Q) = ¬P ∨ ¬Q.

Second Law: ¬ (P ∨ Q) = ¬P ∧ ¬Q
¬ (P ∨ Q) is true when both P and Q are false.
This means both ¬P and ¬Q must be true.
Therefore, ¬ (P ∨ Q) = ¬P ∧ ¬Q.

Implications

For propositions A and B, if A implies B (A → B) then it means that if A is true, then B is true. Since either A can be true or false. “A ∨ ¬A” is true. Which can be written as “B ∨ ¬A”. Since A being false doesn't say anything about B. It may happen that despite A being false, B may be true. Falsity of antecedent (A) doesn't determine anything about consequence (B). This fact is commonly said as false implies anything (true or false). Also, true only implies true. E.g., If you kill a mosquito then it dies. Here A is “you kill a mosquito” and B is “it dies”. If you kill a mosquito, surely it dies, but if you don't kill a mosquito, then the mosquito may or may not die.

Biconditional Propositions: They are defined as "If P then Q, and If Q then P" or "P if Q, and Q if P" or P iff Q. (P ↔ Q) = ((P → Q) ∧ (Q → P)).

Inverse of Implication: (¬P → ¬Q) = (P ∨ ¬Q) = (¬Q ∨ P) = (Q → P)

Biconditional Equivalence: (P = Q) = ((P → Q) ∧ (¬P → ¬Q)) = (P ↔ Q)

Modus Ponens: ((P → Q) ∧ P) → Q
((P → Q) ∧ P) = ((Q ∨ ¬P) ∧ P)
(((Q ∨ ¬P) ∧ P) → Q) = ¬((Q ∨ ¬P) ∧ P) ∨ Q
From De Morgan's Law:
= ¬(Q ∨ ¬P) ∨ ¬P ∨ Q
= ¬(Q ∨ ¬P) ∨ (Q ∨ ¬P). This is a tautology.

Modus Tollens: ((P → Q) ∧ ¬Q) → ¬P
((P → Q) ∧ ¬Q) = ((¬P ∨ Q) ∧ ¬Q)
(((¬P ∨ Q) ∧ ¬Q) → ¬P) = ¬((¬P ∨ Q) ∧ ¬Q) ∨ ¬P
From De Morgan's Law:
= ¬(¬P ∨ Q) ∨ Q ∨ ¬P
= ¬(¬P ∨ Q) ∨ (¬P ∨ Q). This is a tautology.

Fallacy of Converse: (Q ∧ (P → Q)) → P
(P → Q) = (¬P ∨ Q)
((Q ∧ (¬P ∨ Q)) → P) = ¬(Q ∧ (¬P ∨ Q)) ∨ P
From De Morgan's Law:
= ¬Q ∨ ¬(¬P ∨ Q) ∨ P
= ¬Q ∨ (P ∧ ¬Q) ∨ P
= ¬Q ∨ P ∧ (¬Q ∨ True)
= ¬Q ∨ P ∧ ¬Q.
= ¬Q ∧ (P ∨ True)
= P ∧ ¬Q
This is not a tautology.

Fallacy of Inverse: (¬P ∧ (P → Q)) → ¬Q
(P → Q) = (¬P ∨ Q)
((¬P ∧ (¬P ∨ Q)) → ¬Q) = ¬(¬P ∧ (¬P ∨ Q)) ∨ ¬Q
From De Morgan's Law:
= P ∨ ¬(¬P ∨ Q) ∨ ¬Q
= P ∨ ¬Q ∨ (P ∧ ¬Q)
= P ∨ ¬Q ∧ (True ∨ P)
= P ∨ (¬Q ∧ P)
= P ∧ (¬Q ∨ True)
= P ∧ ¬Q
This is not a tautology.

Sufficient Conditions - P → Q (if P then Q). If P is true then Q is true. P is hence a sufficient condition for Q.

Necessary Conditions - P → Q (if P then Q) = ¬Q → ¬P. If Q is false then P is false. Q is hence a necessary condition for P.

The conjunction of some necessary conditions is also a necessary condition:
((P → A) ∧ (P → B))
= (A ∨ ¬P) ∧ (B ∨ ¬P)
= (A ∧ B) ∨ (A ∧ ¬P) ∨ (B ∧ ¬P) ∨ (¬P)
= (A ∧ B) ∨ (¬A ∨ P) ∨ (¬B ∨ ¬P) ∨ (¬P)
= (A ∧ B) ∨ (¬P) = P → (A ∧ B)

Self-Referential Statements

Self-affirming statements - Consider “this statement is true.” This statement is meaningless since if it is true, then it is true, or if it is false, then it is false. The implication is true (valid) but the statement is meaningless since we know that for any A, A → A is always true irrespective of the truth value of A. But a self-affirming statement can be either true or false, hence is not a proposition.

Self-negating statements - Consider “this statement is false.” This statement is also meaningless since if it is true, then it is false, or if it is false, then it is true. The implication is contingent on the truth value of the statement. The self-negating statements are neither true nor false, hence are not propositions.

Consider:

  • If Paris is in France then French people are from France (true implies true).
  • If Paris is in England then English people are from France (false implies false).
  • If Paris is in England then French people are from France (false implies true).

The problem here is that the implications are irrelevant. True implies true, but there is no connection between antecedent and consequent.

First-order logic solves the above problem by allowing for the formalization of statements about entities and their properties. A proposition is resolved into a subject (description of entities) and predicates (description of properties). A predicate is a part of propositions that talks about a property of some variable (an instance of the subject). The meaning of a predicate is exactly a function from the domain of variables to the truth-values “true” and “false”. A domain is the set of all variables for which the property is defined. A domain is necessary to determine the truth value of any predicate. A property is not defined for variables outside its domain.

In checking the consistency and causal relevance of implications, first-order logic uses quantification.

Quantification - This ensures that statements apply either universally or exist under certain conditions. For instance, in an implication P(x) → Q(x), first-order logic requires that there be a consistent domain of discourse. The predicates P and Q must involve entities within the same domain (i.e., x should refer to the same entity in both cases).

Quantifiers - Quantifiers tell you for how many elements in a collection (that take place as variables) the given predicate is true or false.

Universal quantifier - “for all” (∀) - True when predicate is true for all variables.
∀x(P(x)) = P(x1) ∧ P(x2) ∧ P(x1) ∧ ...

Existential quantifier - “there exists” (∃) - True when predicate is true for at least one variable.
∃x(P(x)) = P(x1) ∨ P(x2) ∨ P(x1) ∨ ...

From De Morgan’s Laws:
∼∃x(P(x)) = ∀x(∼P(x))
∼∀x(P(x)) = ∃x(∼P(x))

If a property P is true for all x, then P is an identity of x. If an identity exists, then it is necessary for all x but it is not sufficient. Being a mammal is an identity of cats. Cats are necessarily mammals, but that's not the sufficient condition to be a cat.

An example of first-order proof

Proof that if a proposition p has a finite number of necessary conditions, then the conjunction of those necessary conditions is sufficient for p:

  1. Consider a and b to be necessary conditions for p.   [ (p → a) ∧ (p → b) ]
  2. Consider no other proposition c that is necessary for p.   [ ¬∃c(p → c) ]
  3. Then let us assume that a and b are true but p is false. (p → a) ∧ (p → b) is True. (false can imply true). It means that even if a and b are true, p can be false since they are not sufficient.
  4. If p is false and c is false, then p → c holds true. This should not be the case.
  5. If p is false and c is true, then again p → c holds true. This should not be the case.
  6. If p is true and c is true, then again p → c holds true. This should not be the case.
  7. If p is true and c is false, then only p → c holds false. Which is the condition (2).

So with condition (2), p is true while a and b are true, so a and b together is a sufficient condition for p.

((p → a) ∧ (p → b) ∧ ¬∃c(p → c)) → ((a ∧ b) ↔ p)

Types of Arguments

  • Deductive - Special conclusion from general rule. As long as the general rule is true, a valid special conclusion is also true; hence the deduction is sound.
    ∀x(P(x)) → ∃x(P(x))
  • Inductive - Probable general conclusion from special observation.
    ∃x(P(x)) → ∀x(P(x))
  • Abductive - Probable cause of conclusion from a general rule (making the most likely explanation based on evidence).
    (p → q) ∧ q → p

A table highlighting different types of argument in a syllogism used famously by Peirce.

Abduction Deduction Induction
Case from Rule and Result Result from Rule and Case Rule from Case and Result
Rule (first principle) All the beans in this bag are white Rule (first principle) All the beans in this bag are white Case (hypothesis): These beans are from this bag
Result (conclusion): These beans are white Case (hypothesis): These beans are from this bag Result (conclusion): These beans are white
Case (hypothesis): These beans are from this bag. (The beans were taken out of this bag) Result (conclusion): These beans are white. (These beans that we have now are white) Rule (first principle) All the beans in this bag are white. (all the beans taken out will be white)

The idea of non-deductive arguments introduces the concept of probability.

Only deductive reasoning guarantees the truth of the conclusion. Abductive and inductive reasoning are trial fits to make the deduction work.

In abduction, we seek a probable case that makes the conclusion deductively true given the rule is true. In induction, we seek a probable rule that makes the conclusion deductively true given the case is true. In a way, abduction is about finding what probably happened, and induction is about what probably will happen. Deduction is finding what happens.

Proof by induction has no induction in it. The inductive hypothesis "p(x0)" and the inductive step "p(xk) → p(xk+1)" are proven, and then by repeated modus ponens, p(x) is proven for all x. It is all deductive.

Münchhausen Trilemma

There are only three ways of completing an argument:

  1. The circular argument, in which the proof of some proposition presupposes the truth of that very proposition.
  2. The regressive argument, in which each proof requires a further proof, ad infinitum.
  3. The dogmatic argument, which rests on accepted precepts that are merely asserted rather than defended.

The trilemma, then, is the decision among the three equally unsatisfying options. Karl Popper’s suggestion was to accept the trilemma as unsolvable and work with knowledge by way of conjecture and criticism. This is the essence of science. We do not seek the “ultimate truths” but rather we approximate reality for our uses. If we find errors, we jump to a new, better approximation.

The goal of science is to create a description of the system (The part of reality you are interested in studying) by establishing relations among the properties of the system by knowing all of its properties. At any instant such a description is called a theory. The theory does not exactly describe the system but a model of the system which is an approximation of it. Every observation of a system may give us knowledge of some properties of the system. The model has to be developed such that it explains all the known properties. But such a model is not complete; it has to predict new properties. If such predictions are observed then such observations become evidence for the theory. If observations are contrary to the predictions then the theory is revised or discarded. This process has to be repeated and be called science.

In first order logic, atomic propositions are descriptions of some entity and its properties. An atomic proposition is true if it corresponds to reality or is coherent with other truths or definitions. There are two types of atomic propositions:

1. Analytic

Since the subject is the description of an entity, it requires some sufficient properties that hold for the entity to be labelled as the subject. If there exists an x such that (all definitive properties are true for x), then x is labelled as whatever it is called. They do not necessarily conform to reality but are true because the definitive properties must not contradict each other (coherent truth). The predicate is hence the definition of the subject. So these propositions are true because of coherence. They are definitive statements. They are a priori as they do not require empirical evidence. They are contingent but only on the consistency of the definitive properties.

Examples:

  1. “A quadrilateral has four sides.” (Quadrilaterals are defined as planar shapes which have four sides.)
  2. “A sister is a female sibling.” (“Female siblings” and “sisters” are literally the same thing.)

2. Synthetic

The predicate is not the definition of the subject, where properties other than definitive properties are claimed. They are contingent either on reason or observation:

2.1. A-posteriori Synthetic

Without using any sensory information or inferential experiment, the truth value of these claims cannot be decided. Observation is sufficient to be true. If definitive properties are consistent with the synthetic properties, then it is possible to be true. So looking for observations is motivated. If definitive properties are inconsistent with synthetic properties, then it is impossible to be true, hence false.

So synthetic a-posteriori truths are true by only observation (empirical truths) and false by only inconsistency. Hence the truth value is contingent both on correspondence and coherence.

Examples:

  1. “It is raining.” - To be true, one must observe rain directly, but to be always false, the definition (cloud pouring water on the ground in a certain region) must be inconsistent with the synthetic claim which is “happening”. One of the definitive properties required to identify rain is clouds; if we observe the sky to be clear, we can then claim that “it is raining” is false without observing rain.
  2. “Water is H2O.” - Water is defined as a transparent liquid inorganic substance that humans can drink, but to claim that it is H2O chemically requires evidence to be true. However, to be false, water is defined to have properties that are inconsistent with the claim of being H2O.
  3. “Pigs can fly.” - The definition of pigs and the property of being able to fly are inconsistent, so without observing all the pigs, we can know that there are no pigs that can fly.

Now consider "There is a hippopotamus in this matchbox." I can try to observe but cannot find it, and then it can be argued that it is actually hidden maybe, because not observing the hippos does not make the claim false. But one can take the definition of a hippopotamus and a matchbox and analyze that it is bigger than a matchbox, so it cannot be in this matchbox. So here, without observing a hippo, we can say that the claim is false. In a way, we claimed that “A hippo cannot be in a matchbox” to be true, which is synthetic but not a posteriori.

2.2. A-priori Synthetic

No need for direct experience or observation to be true. Example: “For a triangle, the sum of the angles is 180 degrees.” This is not the definition of a triangle but still is true without any direct observation. However, it is impossible to prove the angle sum property of triangles without having the notion of space. To define something, all sufficient properties must be met. After that, one can find necessary properties. Reason is needed but also some sufficient background experience (conceptual) and definitions.

Consider “ought implies can.” It is not analytic and not a posteriori (though it can be verified), but to be true, it requires reason alone starting from the definitions. Ought to do something is a property of actions that are moral. If someone cannot do an action, then the action is not there, hence cannot be judged for its morality.

Now consider that all synthetic a priori claims are true in their conceptual framework, so the cases where the conceptual framework does not work, the synthetic a priori claims do not work either. For example, the angle sum property is true for triangles in Euclidean geometry only. Hence, a priori synthetic claims are true only coherently.

Conclusion

Synthetic a priori claims are not metaphysical truths (opposed to what Kant argued) because their truth is contingent upon the conceptual framework (e.g., Euclidean geometry or classical mechanics) in which they operate. They are coherent within these frameworks (models of reality or definitions of concepts) but not necessarily true of reality in an absolute metaphysical sense.

Synthetic a posteriori claims, as emphasized by Hume, are based on empirical observation and inductive reasoning, which makes them contingent truths about the physical world. Since they depend on observation and could be otherwise, they are not truly metaphysical in the sense of being necessary truths about reality. Instead, they describe the physical world rather than the metaphysical foundations of reality.

Metaphysical truths are hence the limit of synthetic a priori truths when they are immune to revision, as the definition on which synthetic a priori truths are contingent, is themselves contingent on how agents observe reality. Metaphysical truths are necessary for experience to be possible, but to reach them one has to observe reality and revise the analytic truths. So we can come up with metaphysical truths abductively and then only test them. This is why metaphysics is “analysis” of reality since we set up analytic claims and derive synthetic a priori claims from them and test them like synthetic a posteriori.

Epistemology - The study of how agents interact with reality, theoretically. It is the study of how the agent knows the reality. It is the analysis of knowledge. Knowledge is justified true beliefs without luck. Hence, knowledge is proven true belief. Proof is a stricter standard than justification, which eliminates the problem of epistemic luck by empirical demonstration. JTB (justified true belief) should be formally regarded as informed guesswork.

Ethics - The study of how agents interact with reality, practically. Ethics only applies to moral agents. Moral agents are agents that can make choices. Hence, ethics becomes the study of an agent’s choices. In ethics, the goal is to find the right choices. This defines what the agent ought to do. But ought implies can. Ethics presumes the concepts of free will (for choices) and harm (for morally right choices). Kant said "Ought implies can," but what implies ought? Is it consequences or intentions? And what one is ought to do?
